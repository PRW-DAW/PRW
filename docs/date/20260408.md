## Register

```shell
curl -X POST http://api.devhub.com/api/register \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"name":"Alba","username":"alba","email":"alba@test.com","password":"123456"}'
```

## Login

```shell
curl -X POST http://api.devhub.com/api/login \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"email":"alba@test.com","password":"123456"}'
```
