# API

## Register

```shell
curl -X POST http://api.devhub.com/api/register \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"name":"Ana2","username":"ana2","email":"ana2@test.com","password":"123456"}'
```

## Login

```shell
curl -X POST http://api.devhub.com/api/login \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"email":"ana2@test.com","password":"123456"}'
```
